race_hash (python) v1.0.0
===========================

Python module for Lyra2REv2 PoW hashing algorithm of Race.


Install
-------

Python 2.7+ or 3.5+ is required as well as a gcc.

    $ sudo python setup.py install


Test
-------

After installation, test hash.

    $ python test.py
